# Brain — The Hierarchy of Response

The brain is not a single decision-maker. It is a hierarchy of response systems, each operating at different speeds and with different authorities. Understanding which layer is speaking — and when — is one of the most practical capacities the inner witness can develop.

## The three layers

**Lizard brain** (brainstem and cerebellum) — autonomic regulation: breath, heartbeat, survival reflexes. Operates entirely below conscious awareness. Cannot be philosophised. Can only be supported or undermined by the other systems.

**Amygdala** — the threat-detection and emotional processing centre. Fires before the pre-frontal cortex has been consulted. The amygdala hijack — the moment fear, anger, or urgency overrides deliberate thought — is the most common failure mode of the audit framework. The worth-machine exploits the amygdala: urgency signals run through it, cortisol reinforces it, and the code never gets read.

**Pre-frontal cortex** — the seat of deliberate thought, values, and long-term reasoning. Slowest of the three. When the amygdala fires hard, pre-frontal access is physiologically reduced — the hardware for philosophy goes partially offline. Sleep deprivation, chronic stress, and cortisol saturation all impair pre-frontal function. This is not a character failure. It is biology.

## The interaction effect

The hierarchy is not stable. Chronic cortisol suppresses pre-frontal function over time. Sleep restores it. Physical exercise improves amygdala regulation. The philosophical layer is dependent on the biological layer being in sufficient condition to run it.

## Key inference

The audit framework requires pre-frontal cortex access to run. The amygdala hijack physiologically reduces that access. The double-trigger architecture may need a biological precondition: if the body signal has fired AND cortisol is running hard, the audit may not be executable in that moment. The tripwires must be set in clear-headed time precisely because the hardware goes partially offline under the conditions that trigger them.
