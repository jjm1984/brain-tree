# The Code Layer

*Not yet written. Deliberately.*

The Code layer is the spiderweb beneath the library — a derived set of principles for how to live, decide, and be. It is intentionally emergent, not constructed. It crystallises from the whole once the roots and knowledge are rich enough to generate it honestly.

A partial code looks complete but has dangerous blind spots. The trunk statement serves as the provisional standard until the code emerges naturally:

**Do what is fulfilling — cause no harm.**

For decisions involving harm-to-self, run the four conditions of conscious sacrifice: named, proportionate, preserves capacity, articulable.

The code will be written here when the tree is ready to bear it.
