# The Map Layer

*Generally depends on the Code layer existing first — but see the Map-Layer Exception Principle (meta/principles.md).*

The Map is how the code plays out in practice: daily orientation, decision-point navigation, periodic audit. Most of it cannot be built until the Code layer exists to ground it. The exception: some Map content is a tested, lived, already-functional tool rather than a deduction from Code — that content can be written now, and gets tested against Code over time as Code emerges, rather than waiting on it as a precondition.

Two practices confirmed and written under the exception, 2026-07-11:

## Practice: the seven-step relational repair pattern

A pre-agreed, external repair trigger for withdrawal/pursuit cycles — tested and lived, not theorised. Full content, exposed roots, and open design questions: `layers/02-knowledge/relationships-family/leaf-withdrawal-repair-pattern.md`.

## Practice: the financial reserve, two distinct tools

J explicitly placed this as Map/functional-wisdom, not Roots/philosophy — tested via lived use and observed pattern in others, not derived from abstract principle. No specific dollar figures per J's instruction — stated as 3–6 months of expenses, duration itself explicitly not the point.

**Two distinct tools, not one reserve with one meaning:**

1. **Emergency fund** — absorbs a singular shock, prevents a minor event cascading.
2. **Crisis/reinvention buffer** (3–6 months) — buys OPTIONALITY, specifically the ability to exit an ongoing bad situation (named example: leaving a toxic workplace, now resolved/past for J) without the exit itself becoming the crisis.

Explicitly named as NOT "fuck-you money" (which implies leverage/power over others) — this is leverage over circumstance, the right to act on your own timing, not contempt-based unilateral power.

The borrowed-peace tension (see `leaf-peace-as-ingredient`) is tested and largely resolved in favour of the Stoic "preferred indifferent" reading rather than the fragile-dependency reading — bounded, instrumental, targets a specific named fear rather than vague general unease. J confirmed the pattern observed repeatedly in others (people who stay too long in situations they should leave, purely due to lack of financial buffer) as validating evidence independent of personal case.

**External corroboration:** J named Dave Ramsey and Scott Pape independently as holding the same two-tier structure — this upgrades the practice from grounding-dependent (J's personal tested experience only) toward grounded-standalone, per the portability-field logic (see feedback-loop entry-038) — convergent evidence across independent frameworks, not just personal conviction.

Final "why," resolved by J directly: necessary because he has watched what happens without it in others AND tested it personally and found it valuable — explicitly the observed-pattern-in-others reason, not primarily the personal-peace reason, though both hold.

**Cost named and accepted without negation of value:** years of unrewarded effort building the buffer, possibly never "cashed in" or felt as reward — parallel to the hold-fast/no-dragon material in `leaf-just-cause-motivation-architecture`, though J notably does not rebel against this instance of it the way he did against "outlast" generally. Open, unflagged distinction worth noting: this domain may simply never have been asked to provide aliveness/meaning, only security, so the earlier rebellion doesn't apply here.
