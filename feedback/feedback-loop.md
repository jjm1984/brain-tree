# Brain Tree Feedback Loop

A permanent staging layer. Raw ore — philosophical insights, values observations, knowledge of how J works, gaps, and contradictions — that emerged when the tree was being used against reality. Nothing here is in the tree yet. It is unfinished work awaiting placement, challenge, labelling, and verification.

The provenance field stays permanently. It records what the tree was being used for when the insight appeared.

**Harvest instructions:** For each open entry: assess which branch it belongs to, what leaf state it warrants, and what in the tree supports or challenges it. Only mark as harvested once it has been placed in the tree, tested against existing philosophies, and honestly labelled.
